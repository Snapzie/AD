\section{Task 1}

Vi lader følgende rekursive formel $N(C, i)$ angive antallet af kombinationer, man kan lave med $C$ penge og et udvalg på $p_1, p_2, ..., p_i$ øl:

$$N(C, i) = \begin{cases}
  1 & \text{if } C = 0 \\
  0 & \text{if } C < 0 \text{ or } i = 0 \\
  N(C, i-1) + N(C-p_i, i-1) & \text{else }
\end{cases}
$$
